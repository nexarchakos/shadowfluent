# 🎨 Προσθήκη Logo - Οδηγίες

## 📋 Βήματα:

### Βήμα 1: Ανέβασε το Logo Image

**Το logo image πρέπει να είναι στο `public` folder:**

1. **Ανέβασε το logo image** (π.χ. `logo.png`, `logo.svg`, `logo.jpg`)
2. **Βάλτο στο folder:** `english-shadowing-app/public/`
3. **Όνομα αρχείου:** `logo.png` (ή `.svg`, `.jpg` - ό,τι format είναι)

**Παράδειγμα:**
```
english-shadowing-app/
  └── public/
      └── logo.png  ← Εδώ!
```

---

### Βήμα 2: Ενημέρωση του Code

**Μετά το upload, θα ενημερώσω το `App.tsx` να χρησιμοποιεί το logo.**

---

## 🎯 Recommended:

**Για καλύτερη ποιότητα:**
- **SVG** (scalable, μικρό file size)
- **PNG** (με transparency)
- **JPG** (αν δεν χρειάζεται transparency)

**Μέγεθος:**
- **Width:** ~150-200px (για desktop)
- **Height:** Auto (proportional)

---

**Ανέβασε το logo image στο `public` folder και πες μου το όνομα του αρχείου!** ✅
