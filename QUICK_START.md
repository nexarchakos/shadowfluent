# Quick Start Guide

## Γρήγορη Εγκατάσταση

1. **Εγκατάσταση dependencies:**
   ```bash
   cd english-shadowing-app
   npm install
   ```

2. **Εκκίνηση development server:**
   ```bash
   npm run dev
   ```

3. **Άνοιξε τον browser:**
   - Θα δεις ένα URL τύπου `http://localhost:5173`
   - Άνοιξε το στον browser σου

## Πώς να χρησιμοποιήσεις την εφαρμογή

### Βήμα 1: Επίλεξε Κατηγορία
- Κάνε click σε μια από τις 4 κατηγορίες:
  - **Business English**: Φράσεις για επιχειρηματικό περιβάλλον
  - **Travel**: Φράσεις για ταξίδια
  - **Sport**: Φράσεις για αθλητικά θέματα
  - **Meetings**: Φράσεις για συναντήσεις

### Βήμα 2: Επίλεξε ή Δημιούργησε Φράση
- **Επίλεξε** μια φράση από τη λίστα
- **Δημιούργησε** νέα φράση με AI (κουμπί "Νέα Φράση (AI)")

### Βήμα 3: Ρύθμισε τις Επιλογές
Κάνε click στο "Ρυθμίσεις" και ρύθμισε:
- **Αριθμός Επαναλήψεων**: 1-10 (προεπιλογή: 3)
- **Χρονικό Διάστημα**: 2-30 δευτερόλεπτα (προεπιλογή: 5)
- **Φύλο Φωνής**: Άνδρας ή Γυναίκα
- **Προφορά**: Βρετανική ή Αμερικάνικη

### Βήμα 4: Ξεκίνα το Shadowing
- Κάνε click στο "Έναρξη"
- Η φωνή θα διαβάσει τη φράση
- Περίμενε το countdown
- Επαναλάμβανε τη φράση (shadowing)
- Η διαδικασία θα επαναληφθεί αυτόματα

### Προαιρετικά: Ανέβασε Δικό σου Αρχείο
- Κάνε click στο "Ανέβασε δικό σου αρχείο"
- Επίλεξε ένα `.txt` αρχείο
- Κάθε γραμμή = μια φράση

## Tips

- 🎧 Χρησιμοποίησε ακουστικά για καλύτερη εμπειρία
- 🗣️ Μίλα δυνατά όταν κάνεις shadowing
- ⏸️ Μπορείς να κάνεις παύση αν χρειάζεσαι χρόνο
- 🔄 Χρησιμοποίησε "Επαναφορά" για να ξεκινήσεις από την αρχή

## Troubleshooting

**Δεν ακούγεται η φωνή:**
- Έλεγξε τον ήχο του browser
- Κάνε allow permissions για audio
- Δοκίμασε διαφορετικό browser (Chrome, Firefox, Safari)

**Η φωνή δεν αλλάζει:**
- Οι διαθέσιμες φωνές εξαρτώνται από το browser και το OS
- Δοκίμασε διαφορετικό browser για περισσότερες επιλογές

**Το countdown δεν λειτουργεί:**
- Refresh τη σελίδα
- Έλεγξε αν το JavaScript είναι ενεργοποιημένο
